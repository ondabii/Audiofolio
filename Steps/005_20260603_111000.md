## 수행 작업

### 1. AudioEngine — Web Audio API → `<audio>` 엘리먼트 전환
- **원인**: AudioContext autoplay 정책(특히 Safari)으로 인해 suspended 상태 문제 반복 발생
- **해결**: HTML5 `<audio>` 엘리먼트로 완전 교체
  - `audio.play()` / `audio.pause()` — 단순하고 모든 브라우저에서 동일 동작
  - `audio.loop = true` — 자동 루프
  - `audio.currentTime = t` — Seek
  - `requestAnimationFrame` 루프로 `globalCurrentTime` 동기화

### 2. 파형 시각화 (TrackItem.tsx)

- `items-center`로 수직 중앙 정렬 (이전 `items-end` → 수정)
- 대표 버전: 주황색, 일반: 흰색 반투명

### 3. 신규 업로드 시 파형 추출 (audioUtils.ts)
